## 代码执行模式 — acm

+ acm如何输入与输出

```
一行输入读取：

line = readline()



多行输入读取

let K = parseInt(readline()); // 读取行数 
let inputs = [];

for(let i = 0;i < K; i++){ 
    line = readline()
    let p = line.split(' ');
    inputs.push(p) 
}



输出：

print()

或

console.log()
```
