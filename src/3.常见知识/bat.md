## 打印日志

语法：

```
@echo (打印内容)
```

## 执行git命令

```
@echo -----------------------------------
@echo       一键提交代码
@echo -----------------------------------
@echo 执行开始！
@echo 
git pull
git add .
git commit -m "代码更新！"
git push
@echo 执行结束！
pause
```

适用场景：**多仓库同时提交**，就要用到**脚本工具**来提交！

## pause

停留在窗口不退出，只有按任意键才能退出！
