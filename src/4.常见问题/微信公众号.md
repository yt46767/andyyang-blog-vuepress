## 解决微信内置浏览器无法动态更新title的问题
```javacript
let iframe = document.createElement("iframe")
//设置标题
document.title = '这里是要修改成的新标题';
//随便加载点什么
iframe.src="/favicon.ico"
document.body.appendChild(iframe)
//刷新后移除iframe
setTimeout(function() {
      document.body.removeChild(iframe)
  }, 0);
```
