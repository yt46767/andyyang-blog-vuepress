### CSS3 新增属性

1.**边框**

+ border-radius
+ box-shadow

2.**背景**

+ background-size
+ background-origin

3.**2D**

+ transform

  它可做旋转 `rotate(1turn)`，缩放`scale(1,2)`，倾斜`skew(45deg,45deg)`，平移`translate(10px,10px)`

4.**3D**

+ transform-origin

  以 元素的中心点 变形 `transform-origin: x-axis y-axis z-axis;`

+ rotate3d

  `rotate3d(x,y,z,angle);`

5.**动画**

+ animation
