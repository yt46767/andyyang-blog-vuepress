# console 分级别 webpack 配置 去除 开发 配置

//打包环境去掉console.log
new UglifyJsPlugin({
  uglifyOptions: {
    compress: { // 压缩
        // warnings: false, // 控制台是否显示压缩器的警告信息，默认为否false
        drop_console: false, //不移除console
        drop_debugger: true, //移除debugger
        pure_funcs: ['console.log'], //移除开发日志console.log
    },
    sourcemap:false,
  },
})

注：

（1）drop_console -- 默认 false. 传true的话会干掉console.*函数。如果你要干掉特定的函数比如console.info ，又想删掉后保留其参数中的副作用，那用pure_funcs来处理吧。

（2）sourcemap 编译后代码对源码的映射，用于网页调试
