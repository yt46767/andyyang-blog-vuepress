# html 禁止页面缓存

现在，采用：
--------------------------------------------------------------------------------
在html里的head标签中加
 <meta http-equiv ="proma" content = "no-cache"/>
 <meta http-equiv="cache-control" content="no cache" />
 <meta http-equiv="expires" content="0" />

为什么要禁用页面缓存呢？
--------------------------------------------------------------------------------
用户后退到之前的编辑页，编辑页必须是空白的，才对！
所以才要禁用页面缓存。
